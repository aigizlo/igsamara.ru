# InvestGroup Samara - Сайт-визитка

Современный сайт-визитка компании по недвижимости.

## Запуск

1. Откройте `index.html` в браузере
2. Или используйте Live Server в VS Code

## Структура

- `index.html` - основная страница
- `css/` - стили
- `js/` - скрипты
- `images/` - изображения
- `logo/` - логотип
